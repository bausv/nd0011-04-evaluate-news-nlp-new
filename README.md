# Udacity ND FED - Project 4

## Prerequisites, Installation, Execution
* Node 18 installed
* run `npm install`
* create a `.env` file in the root folder with an entry API_KEY=\<your MeaningCloud API key>
* execute `npm run start` or `npm run start-dev`
